Selectors in CSS

There are 3 selectors in CSS

  1. Element selector
  2. id selector
  3. class selector

1.Element selector:

-->In css the Element selector is add to a particular Element. 

  Example.
                 P{
                    text-align:center;
                    color: red;
                }
                in the above Example Element is the p tag.

2.id selector:

-->In CSS id selector is add to any element but in html language id is unique, so you give unique names to id . 

For Example.
            <p id="paral">My name is manu</p>

                     #paral{
                        text-align:center;
                        color :green;
                     }
                    in the above example we taken id name is paral.
                    so using id attribute we gave name.
                    we can add style to this using # symbol and id name like above example we get the id selector.

3.class selector:

-->In CSS class selector is add to any element but in html language class name is not unique, so you give same names to class name . 

For Example.
                <p class="center">My name is manu</p>

                     .center{
                        text-align:center;
                        color :green;
                     }
                    in the above example we taken class name is paral.
                    so using class attribute we gave name.
                    we can add style to this using . symbol and class name like above example we get the class selector.
