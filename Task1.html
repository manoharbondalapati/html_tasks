Meta tags

Defination:

The meta tag defines metadata about an html Document.Metadata is data about the data.It is always go inside the head tag.

    -->  Example & syntax : <meta> 
    
Attributes

    
1.charset:
 charset attribute specifies the character encoding for the HTML Document. The HTML5 specification encourage web developers to use the UTF-8 character set,
    which covers all of the characters and symbols in the world.

    -->  syntax : <meta charaset="UTF-8">
    

    
2.Name:
 The name attribute specifies the name for the metadata. The name attribute specifies a name for the information / value 
    of the content attribute.
    There are 4 names in name attribute
                                       1.viewport
                                       2.description
                                       3.keywords
                                       4.author.
    -->    syntax : <meta name="" >
    
    1.viewport:
       this is used to appears a good wherever see or in any devices.
         -->Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
         Note: Content attribute is used to specifies value of name attribute.


    2.description:
       this is used to describes clearly what is our web page.
        --> Example: <meta name="description" content="free web tutorials">
        
    3.keywords:
       the meta keywords are the meta tags that can use to give search engines more information about a page's contents.
    -->Example:  <meta name="keywords" content="HTML, CSS, Javascript"> 
    
    4.author:
       the author attribute is used to indicate who is author of page.
    -->Example: <meta name="author" content="Manohar">
    

    
3.http-equiv:

     This is used to refresh the webpage for every 5,10,15... whatever time you need it will refresh.
--> Example:  <meta http-equiv="refresh" content="500">
