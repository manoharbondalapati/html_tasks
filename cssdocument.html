CSS Document

There are ways to write a code in CSS.

1.inline CSS

In innline CSS we write a code at that particular element.

Exmaple:<p Style= "color:red;">this a inline CSS Exmaple</p>

so, we write code like this using style attribute at opening tag of element.

2.internal CSS

In internal CSS we write CSS code at html file using <Style></style>tag.

 
Example:<style>
    h1
       {
         colore : red;
       }
       </style>

3.External CSS

In External CSS we write a code in new file like new CSS file.

Example:
