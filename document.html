Definations Of HTML Concepts

1.Tags

Whatever we written inside the angular brackets are called tags.

2.Elements

Combination of opening tag ,closing tag and content is called element

3.Self closing tag

Some kind of tags are not need to be closed manually by it's closing tag are called self closing tags

4.Void Elements

All self closing elements are called void elements

5.Block Level Elements

Always starts on new line the browsers automatically add some space before and after the element.

6.Comments

Comments is used to understand the code what you written
