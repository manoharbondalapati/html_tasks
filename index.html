Java Script Data Types

There are 2 types of data types in javascript.
                   1.primitive Data types.
                   2.non-primitive Data types.

Primitive Data Types

1.number

            ->All the JavaScript numbers are stored as decimal numbers(floating point).
            ->Numbers can be written with, or without decimal.
            ->In JavaScript 2^53 is last number.

                Example:
                    var num1=900719925740992;
                    console.log(num1);

2.String

            ->A String(or a text string) is a series of characters like "manohar".
            ->Strings are written with quotes. You can use single or double quotes.
            
                Example:
                    var str1="manoahr";
                    console.log(str1);

3.bigint

            ->JavaScript bigint is a new datatype that can be used to store values that are too big 
             to be represented by a normal javascript NUmber.
                            
                Example:
                        var num1=bigint(44585558416387984649797469414197416);
                        console.log(num1);

4.boolean

            ->JavaScript boolean same as other programming languages true or fasle .
                                
                Example:
                        var iscoding =true;

5.null

            ->JavaScript in null is only one value in case declaring value with null.
                                                
                 Example:
                        var age =null;

6.undefined

            -> in JavaScript undefined is a defined as a variable that has not been 
            assigned a value is undefined.
                                                                
                                 Example:
                                        var age;--> is undefined

7.symbol

            -> in JavaScript symbol is unique and that can be used as the key of an 
            object property.
