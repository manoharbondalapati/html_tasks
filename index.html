Object Methods

1.Object.keys()

          Object.keys method is used to display only object keys
               for Example     var employee = {id : 12 , name : "manu"};
                                   console.log(Object.keys(employee));

2.Object.values()

          Object.keys method is used to display only object values
               for Example     var employee = {id : 12 , name : "manu"};
                                   console.log(Object.values(employee));

3.Object.entries()

          Object.keys method is used to display both keys and values in array format
               for Example     var employee = {id : 12 , name : "manu"};
                                   console.log(Object.entries(employee));

4. .hasOwnProperty()

          this method is used to find wheather the key is present or not in the object
               for Example     var employee = {id : 12 , name : "manu"};
                                   console.log(employee.hasOwnProperty("id"));

5.Object.assign()

          this method is used to copy the values and properties from one or more objects
               for Example     var employee = {id : 12 , name : "manu"};
                               var student  = {role: "engineer"}
                               var details = Object.assign({} , employee, student);
                               console.log(details);

6.Object.freeze()

           when this method is wrote cannot change, update, add new properties and values into an object
               for Example     var employee = {id : 12 , name : "manu"};
                                   Object.freeze(employee);

6.Object.seal()

           when this method is wrote cannot change, add new properties and values into an object but 
           update the properties values.
               for Example     var employee = {id : 12 , name : "manu"};
                                  console.log(employee);
                                var student = Object.seal(employee);
                                employee.name = 'manohar';
                                    console.log(student);
